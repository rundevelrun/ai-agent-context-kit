# 🤖 ai-agent-context-kit

AI 에이전트와 최적의 협업을 하기 위한 **개인화된 컨텍스트 지침** 저장소입니다.

> ⚠️ **현재 상태:** 규칙 모듈화 및 정리 작업이 진행 중입니다. 모든 공통 규칙은 현재 `rule/COMMON.md`에 통합되어 있습니다.

---

## 🎯 핵심 원칙 (Core Principles)

| 구분 | 지침 | 비고 |
| :--- | :--- | :--- |
| **언어 및 태도** | 정중한 한국어 사용, 불필요한 인사 생략, 핵심 위주 답변 | 소통 효율화 |
| **보안 및 안전** | 민감 정보 하드코딩 절대 금지, 보안 및 확장성 최우선 고려 | 최우선 가치 |
| **보고 체계** | 작업 완료 후 수정 파일 목록 및 주요 변경 사항 요약(표 활용) 제공 | 가시성 확보 |
| **의사 결정** | 임의 판단 지양, 모호한 사항은 반드시 사전 질문 및 확인 후 진행 | 리스크 관리 |

## 🚀 에이전트 사용 지침 (For AI Agents)

1. **규칙 숙지:** 모든 작업을 시작하기 전 `rule/COMMON.md` 파일을 읽고 정의된 원칙을 내부 메모리에 로드하십시오.
2. **비즈니스 로직 이해:** 코드 수정 시 '무엇(What)'을 했는지보다 '왜(Why)' 그렇게 했는지 논리적 근거를 먼저 설명하십시오.
3. **작업 세분화:** 요청받은 작업을 1일 이내 완료 가능한 단위로 쪼개어 계획을 먼저 제시하고 확인받으십시오.
